# sonchain
Sonchain یک اسکریپت در حال توسعه است که با ترکیب نرم افزار های wrapper قدرتمند , و ابزار های vpn-proxy داخلی ( فعلا tor ) یا خارجی , و ابزار های دیگر امکان اجرای دستورات , برنامه‌ها و... از طریق پروکسی و رفع محدودیت‌های ارتباطی مبتنی بر IP , DNS و جغرافیایی و قابلیت های گسترده تر دیگر را فراهم می‌کند.
